## Getting Started

### Prerequisites

* npm
```sh
npm install npm@latest -g
```

### Installation

1. Clone the repo
```sh
git clone https://github.com/ClintonEnyinna/old-apple-page-clone.git
```
2. Go to the app directory
```sh
cd old-apple-page-clone
```
3. Open with visual studio code
```sh
code . 
```
